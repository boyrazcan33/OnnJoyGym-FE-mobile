# OnnJoy - Fitness Community Platform

## 👤 Founder & Developer

- **Ugur Can Boyraz** - Software Developer & Founder

## 📖 Project Description

OnnJoy solves three core problems for gym-goers in Estonia: uncertainty about which gym to choose, loneliness while training, and lack of structured progress.

A comprehensive fitness community platform that provides:
- Expert gym reviews from an 11-year NCAA-certified trainer
- Buddy matching system to find training partners
- Video-verified leaderboards for competition
- Personalized workout programs through goal-based clubs

## 🛠️ Tech Stack

**Backend:**
- Java 21, Spring Boot 3
- Spring Security, JWT Authentication
- PostgreSQL, Flyway
- AWS S3, AWS Rekognition
- Docker, AWS Lightsail

**Frontend:**
- Angular 18, Angular Material
- Server-Side Rendering (SSR)
- Capacitor (iOS & Android)
- Deployed on Vercel

**Testing:**
- JUnit 5, Mockito, Testcontainers
- RestAssured, Jasmine, Playwright, Cucumber
